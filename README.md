# 🌍 Aarya Travels Website

A modern and responsive **Travel Agency Website** built using HTML, CSS, and JavaScript.
This project showcases a complete multi-page travel website with a clean UI and user-friendly design.

---

## ✨ Features

* 🏠 Home Page with Hero Section
* ℹ️ About Page (Company Information)
* 🌍 Destinations Page (Top travel places)
* 💼 Packages Page (Travel plans & pricing)
* 📞 Contact Page (Form with validation)
* 📱 Fully Responsive Design
* 🎨 Modern UI with hover effects
* ✈️ Travel-themed branding

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
